# 플레이봇 검 자동 강화 봇

> ⚠️ **면책 조항**: 이 프로그램은 **교육 및 학습 목적**으로만 제작되었습니다. 자동화 프로그램 사용은 게임 이용 약관을 위반할 수 있으며, 이로 인한 계정 정지나 기타 불이익에 대해서는 **일체 책임지지 않습니다**. 사용 시 본인 책임 하에 이용해 주세요.

## 소개

카카오톡 플레이봇의 "검 키우기" 게임을 자동화하는 GUI 프로그램입니다.

## 주요 기능

- 🎯 **자동 강화**: 목표 레벨까지 자동으로 강화
- 💰 **스마트 판매**: 무기 종류(일반/히든)에 따라 최적 레벨에서 자동 판매
- 📊 **실시간 통계**: 판매 횟수와 총 획득 골드를 실시간으로 확인
- 🔍 **상태 감지**: 시작 시 현재 강화 상태를 자동으로 파악
- 🎨 **직관적인 GUI**: 클릭 몇 번으로 모든 설정 완료

## 설치

### 1. Python 설치
[Python 공식 사이트](https://www.python.org/)에서 Python 3.7 이상 설치

### 2. 라이브러리 설치
```bash
pip install -r requirements.txt
```

## 사용법

### 1. 실행
```bash
python kakao_enhance_bot.py
```

### 2. 좌표 설정
- "좌표 설정" 버튼 클릭
- 카카오톡 채팅창 중앙에 마우스를 올리고 Enter
- 입력창에 마우스를 올리고 Enter

### 3. 설정 조정
- **목표 레벨**: 몇 강까지 강화할지 설정
- **판매 설정**: 일반 무기와 히든 무기의 판매 레벨을 슬라이더로 조정
  - 일반 무기 (검/몽둥이/막대): 기본 11강
  - 히든 무기 (젓가락/우산/광선검 등): 기본 9강

### 4. 시작
"시작" 버튼을 클릭하면 봇이 작동을 시작합니다.

## 무기 종류별 판매 전략

| 무기 타입 | 예시 | 권장 판매 레벨 | 이유 |
|---------|------|--------------|------|
| 일반 무기 | 검, 몽둥이, 막대 | 11강 | 11강부터 비싸게 판매됨 |
| 히든 무기 | 젓가락, 우산, 광선검, 핫도그, 칫솔 등 | 9강 | 9강부터 비싸게 판매됨 |

## 실행 예시

```
현재 상태 확인 중...
   ✅ 현재 상태: 히든 무기 +5
봇 시작: 목표 +15, 판매 설정 [일반 11강 / 히든 9강]
>> 강화 명령 전송...
   [응답] 상태: SUCCESS | 레벨: 6 | 무기: 히든
   ✨ 성공! +6
   ...
   [응답] 상태: SUCCESS | 레벨: 9 | 무기: 히든
   💰 히든 무기 9강 달성! 판매합니다...
   💵 판매 완료! 150골드 획득 (총 1회 판매, 150골드 획득)
```

## 배포 (선택사항)

친구들과 공유하고 싶다면 실행 파일(.exe)로 만들 수 있습니다:

```bash
pip install pyinstaller
pyinstaller --onefile --windowed --name "검자동강화" kakao_enhance_bot.py
```

`dist` 폴더에 실행 파일이 생성됩니다. (Python 없이 실행 가능)

## 주의사항

- 봇 실행 중에는 카카오톡 창을 최소화하거나 가리지 마세요
- 마우스와 키보드를 사용하지 마세요 (간섭 방지)
- 골드가 부족하면 자동으로 멈춥니다

## 문제 해결

| 문제 | 해결 방법 |
|------|----------|
| "좌표가 설정되지 않았습니다" | 먼저 좌표 설정을 완료하세요 |
| "응답 없음" 반복 | 카카오톡이 실행 중인지, 플레이봇 대화방이 열려 있는지 확인 |
| 프로그램이 실행 안 됨 | Python과 필요한 라이브러리가 설치되었는지 확인 |
