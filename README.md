# 검 키우기 자동 봇 (초등학생용)

> ⚠️ **면책 조항**: 이 프로그램은 **교육 및 학습 목적**으로만 제작되었습니다. 자동화 프로그램 사용은 게임 이용 약관을 위반할 수 있으며, 이로 인한 계정 정지나 기타 불이익에 대해서는 **일체 책임지지 않습니다**. 사용 시 본인 책임 하에 이용해 주세요.

## 📖 소개
카카오톡 플레이봇의 "검 키우기" 게임을 자동으로 강화해주는 프로그램입니다.
초등학생도 쉽게 사용할 수 있도록 그래픽 화면(GUI)으로 만들었습니다!

## ✨ 주요 기능
- 🎯 원하는 목표 레벨까지 자동 강화
- 💰 **무기 종류별 자동 판매**
  - 일반 무기 (검/몽둥이/막대): 11강부터 비싸게 판매 (조정 가능)
  - 히든 무기 (젓가락/우산/핫도그/칫솔/광선검 등): 9강부터 비싸게 판매 (조정 가능)
  - 슬라이더로 쉽게 판매 레벨 조정 (0~20강)
- � **실시간 판매 통계**
  - 판매 횟수와 총 획득 골드를 화면에서 실시간으로 확인
  - 잠깐 다른 일을 하다가 돌아와도 얼마나 벌었는지 한눈에 파악!
- �🔍 **시작 전 현재 상태 자동 파악**
  - 봇 시작 시 채팅창을 먼저 확인하여 현재 몇 강인지 자동으로 파악
  - 무기 종류도 자동으로 감지 (일반/히든 구분)
- 🖱️ 마우스 클릭으로 간편하게 사용

## 📦 설치 방법

### 1단계: 필요한 프로그램 설치
먼저 Python이 설치되어 있어야 합니다. Python이 없다면 [python.org](https://www.python.org/)에서 다운로드하세요.

### 2단계: 필요한 라이브러리 설치
명령 프롬프트(또는 PowerShell)를 열고 **프로젝트 폴더에서** 다음 명령어를 입력하세요:

```bash
pip install -r requirements.txt
```

또는 직접 설치:
```bash
pip install pyautogui pyperclip
```

## 🚀 사용 방법

### 기본 실행
1. PC 카카오톡을 켜고 플레이봇과 대화방을 엽니다
2. `kakao_enhance_bot.py` 파일을 더블 클릭하거나, 명령 프롬프트에서 실행합니다:
   ```bash
   python kakao_enhance_bot.py
   ```

### GUI 사용 가이드
프로그램 창이 열리면 다음 순서대로 진행하세요:

#### 1️⃣ 좌표 설정 (처음 한 번만)
1. **"1. 좌표 설정 (필수)"** 버튼을 클릭합니다
2. 새로운 작은 창이 뜹니다
3. 카카오톡 창이 보이도록 옆으로 배치합니다
4. **[단계 1]** 마우스를 **채팅 내역(대화창 중앙)** 위에 올립니다
5. **Enter 키**를 누릅니다
6. **[단계 2]** 마우스를 **입력창(글 쓰는 곳)** 위에 올립니다
7. **Enter 키**를 누릅습니다
8. "좌표 설정 완료!" 메시지가 나타납니다

#### 2️⃣ 설정하기
- **목표 레벨**: 몇 강까지 강화할지 설정 (기본값: 15)
- **자동 판매 설정**:
  - ✅ **자동 판매 사용**: 체크하면 자동 판매 기능이 켜집니다
  - **일반 무기 슬라이더**: 검/몽둥이/막대를 몇 강에 팔지 설정 (기본: 11강)
  - **히든 무기 슬라이더**: 젓가락/우산/광선검 등을 몇 강에 팔지 설정 (기본: 9강)
  - 슬라이더를 좌우로 움직여서 0~20강 사이에서 자유롭게 조정할 수 있습니다

#### 3️⃣ 시작하기
1. **"▶ 시작"** 버튼을 클릭합니다
2. 봇이 먼저 채팅창을 확인하여 **현재 상태를 파악**합니다
3. 자동으로 강화를 시작합니다
4. 실행 로그 창에서 진행 상황을 확인할 수 있습니다

#### 4️⃣ 중지하기
- 언제든지 **"⏹ 중지"** 버튼을 클릭하면 현재 작업 완료 후 멈춥니다

## 💡 팁
- **좌표가 잘못되었을 때**: 다시 "1. 좌표 설정" 버튼을 눌러서 재설정하세요
- **응답이 없을 때**: 카카오톡 창이 최소화되어 있지 않은지 확인하세요
- **연속 타임아웃**: 플레이봇이 응답을 안 하면 봇이 자동으로 멈춥니다

## 🎁 초등학생도 쉽게 쓰는 방법 (EXE 파일 만들기)

Python을 모르는 친구들에게 나눠주려면 실행 파일(.exe)로 만들 수 있습니다!

### 1단계: PyInstaller 설치
```bash
pip install pyinstaller
```

### 2단계: EXE 파일 만들기
```bash
pyinstaller --onefile --windowed --name "검키우기봇" kakao_enhance_bot.py
```

### 3단계: 파일 찾기
- `dist` 폴더 안에 `검키우기봇.exe` 파일이 생성됩니다
- 이 파일만 복사해서 다른 컴퓨터에서도 실행할 수 있습니다!
- 단, Python 없어도 실행 가능하지만 **파일 크기가 큽니다** (약 20~30MB)

## ⚠️ 주의사항
1. **자동 프로그램 사용**: 게임 규칙을 확인하고 사용하세요
2. **카카오톡 창**: 봇 실행 중에는 카카오톡 창을 최소화하거나 가리지 마세요
3. **마우스/키보드**: 봇이 실행 중일 때는 마우스와 키보드를 사용하지 마세요 (간섭 방지)
4. **골드 부족**: 골드가 부족하면 자동으로 멈춥니다

## 🐛 문제 해결
- **"좌표가 설정되지 않았습니다" 오류**: 먼저 좌표를 설정해주세요
- **"응답 없음" 반복**: 카카오톡이 켜져 있는지, 플레이봇 방이 열려 있는지 확인하세요
- **프로그램이 안 열려요**: Python이 설치되어 있는지 확인하고, 필요한 라이브러리를 설치했는지 확인하세요
